This directory's series.csv contains ILLUSTRATIVE SAMPLE VALUES, not a real
upstream pull.

It was generated without network access (the build environment could not reach
fred.stlouisfed.org or Yahoo Finance). The numbers are plausible-looking
placeholders so the project builds end-to-end and CI passes; they are NOT the
actual free cash flow, M2, yields, or S&P 500 levels of any company or date.

While this file exists, every rendered chart is watermarked "SAMPLE DATA".

To replace it with real data (on a machine that can reach FRED + Yahoo Finance):

    make data      # fetches real series.csv AND deletes this sentinel
    make           # re-renders the (now un-watermarked) charts

Then commit the refreshed data/ and output/.
